Projeto feito em Node.js
Framework - Express
banco de dados relacional - mySQL


sobre: Um simples Api de encontros de carros, sendo exatamente 2 específicos.
1° Rota de usuário no qual poderá criar seu perfil, tendo as opções: Nome, Email, Telefone e Carro.
